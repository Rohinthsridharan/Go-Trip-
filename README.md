# ✈️ Go Trip — Travel Agency Website

> Your trusted travel companion for unforgettable adventures around the world.

---

## 🌍 Overview

**Go Trip** is a fully responsive, multi-page travel agency website built with pure HTML, CSS, and JavaScript. It allows users to browse curated tour packages, explore destinations, and get in touch with the agency — all through a clean and modern UI.

---

## 📸 Pages

| Page | Description |
|------|-------------|
| `index.html` | Landing page with search, featured tours, testimonials, and about section |
| `tours.html` | Full tours listing page |
| `package.html` | Tour packages overview |
| `details.html` | Individual tour detail page (dynamic via URL query params) |
| `blog.html` | Travel blog and articles |
| `about.html` | About the company |
| `contact.html` | Contact form and info |

---

## ✨ Features

- 🔍 **Destination Search** — Search bar with a datalist of popular destinations (London, Canada, Monaco, France, Japan, Switzerland, Seoul)
- 🗓️ **Date Picker** — Select travel dates from the homepage
- 🃏 **Tour Cards** — Featured packages with destination, price, duration, and ratings
- 💬 **Traveler Testimonials** — Quotes section with customer reviews
- 📋 **Company Info** — Highlights including 8000+ local guides, 28+ years experience, and 98% happy travelers
- 📱 **Responsive Design** — Mobile-friendly layout across all screen sizes
- 🦶 **Footer** — Quick links, new products, and support sections

## 🚀 Getting Started

1. **Clone the repository**
   ```bash
   git clone https://github.com/Rohinthsridharan/Go-Trip-.git
   ```

2. **Open in your browser**
   ```bash
   cd Go-Trip-
   open index.html
   ```

   Or simply drag `index.html` into any modern browser — no build tools or dependencies required.
